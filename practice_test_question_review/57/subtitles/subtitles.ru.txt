Добрый день, уважаемые студенты! Мы с вами начинаем разбор вопроса Practice Test-а. Согласно вопросу, мы являемся компанией по онлайн обучению. Наше приложение написано на Microsoft.NET, хостится на Windows Server машинах на локальном дата-центре. Мы используем стандартную версию Oracle баз данных. Руководство решило мигрировать IT инфраструктуру в облако AWS для того, чтобы стать высокодоступным. В процессе миграции необходимо минимизировать изменения кода, а также необходимо предоставить возможность легкого управления вашей средой инфраструктуры в облаке. Необходимо из предложенных вариантов выбрать два варианта, которые позволяют это все реализовать. Вы видите пять предложенных вариантов ответа. Второй и третий являются правильными вариантами ответов. Обратите внимание, что варианты ответов сильно отличаются друг от друга, поэтому можно отнести этот вопрос к уровню сложности выше среднего. И так, давайте попробуем разобрать каждый вариант ответа и убедиться, почему оно правильное или неправильное. Нам необходимо вспомнить о двух сервисах. Первый - это AWS Database Migration Service, вкратце AWS DMS. Это тот сервис, который позволяет мигрировать локальную базу данных в облако AWS. У него очень много полезного функционала, один из которых называется AWS Schema Conversion Tool или AWS SCT. Благодаря этому функционалу есть возможность сменить движок базы данных. Например, если у вас локальная база данных Oracle, то вы можете, используя этот инструмент, создать аналогичную схему для другого движка, например, MySQL, и уже в облаке AWS продолжать работать с MySQL базой данных. Другой сервис - AWS Elastic Beanstalk. Это тот сервис, который позволяет вам упростить управление вашей инфраструктуры в облаке. Вам необходимо лишь загрузить код вашей программы, вашего приложения, а также на вход подать файл конфигурации будущей инфраструктуры в облаке. Давайте остановимся на вариантах ответов, которые являются неверными. Два варианта ответа, которые содержат контейнеры, это Amazon EKS и Amazon ECS, являются неверными. Согласно вопросу, нам нужно было выбрать опцию, где минимально задействована команда разработки. Здесь же, когда мы говорим про контейнеры, переход на контейнеры требует значительную доработку приложения, поэтому оба эти варианта в рамках этой вопроса являются неправильными. Третий неправильный вариант, связанный с AWS Application Migration Service, также является не совсем подходящим. По той простой причине, у нас есть Oracle баз данных, которую нужно отдельно перенести, и приложение, которое также нужно отдельно перенести в облако. В случае с приложением хорошо подходит Elastic Beanstalk, а в случае с Oracle лучше подойдет приложение, которое специализируется на миграции баз данных, это - AWS DMS. Поэтому вариант с AWS Application Migration Service является неверным. На этом мы разобрали вопрос Practice Test-а. Спасибо за внимание.
