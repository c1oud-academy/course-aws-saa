 Добрый день, уважаемые студенты! Мы с вами начинаем разбору вопроса practice теста. Согласно вопросу нам дается мультинациональная компания занимается разработкой игр для платформ таких как PlayStation, Xbox, Nintendo, также занимается разработкой мобильных игр для Android и iOS. Вы видите, что компания предоставляет достаточно широкий набор продуктов и сервисов, поэтому вы как solutions architect предлагаете им начать использовать API Gateway. Из предложенных вариантов вам необходимо выбрать два варианта, которые являются преимуществами API Gateway и будут полезными для компании. На этом слайде вы видите предложенные варианты ответов, также видите, что первый и второй вариант является верными. Чтобы разобрать каждый вариант ответа, давайте вспомним, что такое Amazon API Gateway. Это managed service, который предоставляет нам возможность создания своего API. При этом этот API может масштабироваться до любых размеров. Преимущественно также является то, что вы оплачиваете только за то, что вы используете. В случае с API Gateway мы оплачиваем за количество вызовов к нашему API, а также объема трафика проведена через наш API. Если говорить про основной функционал API Gateway, он также нам позволяет организовать наш трафик, настроить авторизацию, управлять правами доступа к этим API, встроенный мониторинг присутствует, а также есть возможность управлять различными версиями ваших API. Поэтому первый и второй вариант ответов являются верными. Если мы говорим про оставшиеся три варианта, тот вариант, который содержит GraphQL, он не относится к API Gateway, такой функционал не предоставляется. Если мы говорим про вариант static and unicast IP-адрес, он также не относится к API Gateway, больше относится к сервису AWS Global Accelerator. И самый последний вариант ответа, он также не относится к API Gateway, он больше относится к сервису Elastic Fabric Adapter. На этом мы разобрали вопрос практистеста. Спасибо за внимание.
