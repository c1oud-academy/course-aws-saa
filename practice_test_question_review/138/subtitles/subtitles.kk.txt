Қайырлы күн, құрметті студенттер! Біз Practice Test сұрағын талдауды бастаймыз. Сұраққа сәйкес, бізге PlayStation, Xbox, Nintendo сияқты платформалар үшін ойындар әзірлейтін, сонымен қатар Android және iOS үшін мобильді ойындарды әзірлейтін көпұлтты компания берілген. Сіз компанияның өнімдер мен сервистердің жеткілікті кең спектрін ұсынатынын көресіз, сондықтан Solutions Architect ретінде сіз оларға API Gateway пайдалануды ұсынасыз. Ұсынылған нұсқалардың ішінен API Gateway артықшылықтары болып табылатын және компания үшін пайдалы болатын екі нұсқаны таңдау керек. Бұл слайдта сіз ұсынылған жауап нұсқаларын көресіз, сонымен қатар бірінші және екінші нұсқалардың дұрыс екенін көресіз. Әрбір жауапты талдау үшін Amazon API Gateway сервисін еске түсірейік. Бұл - бізге өз API құру мүмкіндігін беретін managed service. Сонымен қатар, бұл API кез келген өлшемге дейін масштабтауға болады. Тағы бір артықшылығы - сіз тек пайдаланғаныңыз үшін төлейсіз. API Gateway жағдайында біз API-ге қоңыраулар саны, сондай-ақ API арқылы тасымалданатын трафик көлемі үшін төлейміз. Егер API Gateway-дің негізгі функционалдығы туралы айтатын болсақ, ол бізге трафикті ұйымдастыруға, авторизацияны орнатуға, осы API-ге кіру құқықтарын басқаруға мүмкіндік береді, кірістірілген мониторинг бар, сонымен қатар API-дің әртүрлі нұсқаларын басқару мүмкіндігі бар. Сондықтан бірінші және екінші жауап нұсқалары дұрыс. Егер біз қалған үш нұсқа туралы айтатын болсақ, онда GraphQL бар нұсқасы API Gateway-ге қатысты қолданылмайды, мұндай функционалдылық жоқ. Егер біз static anycast IP address нұсқасы туралы айтатын болсақ, ол да API Gateway-де қолданылмайды, ол көбінесе AWS Global Accelerator сервисіне қатысты. Және ең соңғы жауап, ол да API Gateway-ге қатысты емес, ол Elastic Fabric Adapter сервисіне қатысты. Осымен біз Practice Test сұрағын қарастырдық. Назарларыңызға рахмет.
