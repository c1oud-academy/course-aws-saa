Қайырлы күн, құрметті студенттер! Сізді зертханалық жұмысты талдау сессиясында көруге қуаныштымын. Бүгінгі зертханалық жұмыстың тақырыбы - Amazon RDS дерекқор инстансын жасау. Ендеше, бастайық. AWS Management Console негізгі парақшасынан бастайық. Тапсырмаға сәйкес Amazon RDS сервисіне өту керек, содан кейін дерекқор инстансын жасау керек. Ол үшін Create database батырмасын басыңыз. Біз деректерді енгізу парақшасына бағытталамыз. Мұнда Standard Create опциясын таңдау керек. Содан кейін қозғалтқыш ретінде MySQL таңдау керек. Template ретінде Dev/Test таңдау керек. Дәл төменде Availability and Durability бөлімі орналасқан жерде Single DB database таңдау керек. Зертханалық жұмыс үшін Multi-AZ опциясы қажет емес. Жалғыз айырмашылық дерекқордың standby инстансы басқа availability аймағында жасалады. Біздің жағдайда бір дерекқор инстансы жеткілікті, сондықтан осы опцияны таңдап, жалғастырамыз. Settings бөлімінде дерекқор атауын енгізу керек. Біздің жағдайда inventory-db. Master username аты admin болады. Password ретінде зертханалық тапсырмада көрсетілген құпия сөзді енгізу керек. Оны қайталау керек. Әрі қарай, Instance configuration бөлімінде Burstable Classes таңдау керек. Бұл t класстары, дәрісте айтқанымыздай, ең танымал және жүктеменің көптеген түрлеріне жарамды. Дерекқор түрі ретінде t3.micro таңдау керек. Networking параметрлерінде сәл төменірек VPC ретінде Lab-VPC таңдау керек. Security group ретінде барын таңдап, бізге дерекқорлар үшін арнайы жасалған Security Group DB-SG қажет екенін көрсету қажет. Әдепкі опция да осы жерде таңдалады, бірақ оны алып тастау керек. Сәл төменде, Monitoring бөлімі орналасқан жерде Enable Enhanced Monitoring опциясынан құсбелгісін алып тастауыңыз керек, себебі бұл зертханалық жұмыстың аясында қажет емес. Enhanced monitoring опциясы қосылғанда, дерекқорды жасау ұзағырақ болады, сондықтан құсбелгіні алып тастаймыз. Әрі қарай, Additional configuration бөлімін кеңейту керек. Ал Initial database name үшін inventory деп көрсету керек. Enable Automated Backups опциясынан құсбелгіні алып тастау керек. Содан кейін парақшаның соңына қарай жылжып, Create database батырмасын басу керек. Біз дерекқорлар тізімі бар парақшаға бағытталамыз. Қазіргі уақытта бұл жалғыз деректер базасы. Күй Creating болады. 2-3 минут күтуіңіз керек. Refresh батырмасын мерзімді түрде басуға немесе F5 батырмасын басу арқылы бүкіл парақшаны жаңартуға болады. Біраз уақыттан кейін күй Available болады және зертханалық жұмыстың екінші тапсырмасына көшеміз. Біз екінші тапсырмаға көшеміз. Мұнда біз қолданбамызға өтуіміз керек. Қолданбада дерекқор деректерін енгізіңіз және дерекқор мен біздің қолданба арасындағы байланысты баптаңыз. Ол үшін алдымен EC2 сервисіне ауысуымыз керек. Ол үшін сервисті іздеу жолағына EC2 деп енгізіңіз. Бірінші сілтеме - бізге қажет сілтеме. Әрі қарай, сол жақ шарлау мәзірінде Instances опциясын таңдау керек. Біздің уақытша аккаунтымызда бір ғана EC2 инстансы бар екенін көресіз. Ол App Server деп аталады. Дұрыс, бұл бізге керек инстанс. Енді осы инстанстың метадеректерін көру үшін оны таңдауымыз керек. Бұл жерде бізді Details қойындысы қызықтырады. Бізге екінші бағандағы бірінші мән қажет - бұл жалпыға ортақ IP мекенжайы. Сіз оны көшіріп, қолданбаны веб-браузерде бөлек парақшада ашуыңыз керек. Chrome немесе Firefox пайдалану ұсынылатынын ескеріңіз. Қолданба ашылғаннан кейін Settings батырмасын басуыңыз керек. Басқаннан кейін тиісті парақшаға бағытталасыз. Мұнда 4 өрісті толтыру керек. Біріншісі - дерекқорымыздың Endpoint. Әрі қарай, дерекқор аты, пайдаланушы аты және қосылу үшін құпия сөз. Endpoint табу үшін қолданба парақшасын жаппастан, біз AWS Management Console қойындысына оралып, RDS сервисіне өтеміз. Мұнда сол жақ шарлау мәзірінен Database опциясын таңдау керек. Ол осы AWS аккаунтындағы дерекқорлар тізіміне бағыттайды. Біз inventory-db дерекқорын таңдаймыз. Оны басып, дерекқорымыздың негізгі парақшасын ашқанда Connectivity and Security қойындысы ашық болады, өйткені ол бірінші қатарда тұр. Мұнда Endpoint көшіру керек. Ол дерекқорымызға бірегей болады. Сондай-ақ, зертханадағы 25-қадамда толтырылған басқа мәндерді көресіз. Бұл деректер базасы, пайдаланушы аты және құпия сөз. Бұл мәндерді қолданба парақшасына көшіріп, Save батырмасын басу керек. Мұны істегеннен кейін қосымшамыздың басты парақшасына бағытталасыз, оның дерекқорға қосылғанын және онда алдын ала толтырылған үш мән бар екенін көре аласыз. Жаңа мәндерді қосып көрейік. Барлығы 5 не одан да көп жазба жасау үшін кемінде екі мән қосуымыз керек. Сондай-ақ, бұрыннан бар жазбаларды өңдеуге болады. Жазбаны қосу үшін Add inventory батырмасын басып, үш өріске мәндерді енгізу керек. Бұл дүкеннің атауы, заттың атауы және саны. Біз сынақ жазбаларын енгіздік Store 1, Item 1, саны 1 және Store 2, Item 2, саны 3. Жақсы. Осымен біз зертханалық жұмысты аяқтаймыз және автоматты бағалауды іске қоса аламыз. Бірақ ол жерге өтпес бұрын, дерекқормен байланысты барлық деректер Parameter store-да да сақталады деп айтылады. Егер сервистерді іздеу жолағында Features бөлімінде Parameter store деп енгізсек, ең бірінші сілтеме бізге қажет сілтеме. Оған өткенде, мұнда 4 параметр жасалғанын көресіз. Егер сіз олардың әрқайсысын жеке ашсаңыз, бұрын қолданбаға енгізген мәндерді көресіз. Тамаша. Енді автоматты бағалауға көшуге болады. Ол үшін зертханалық жұмыс бар парақшаға оралып, Submit батырмасын басу керек. Бағалауды шынымен іске қосқымыз келетінін сұрайтын хабарлама пайда болады. Иә, біз «Иә» батырмасын басамыз және бірнеше секундтан кейін қорытынды бағаңызды, сондай-ақ тапсырма бойынша ұпайларыңызды көресіз. Біз 20-дан 20 ұпай жинағанымызды көреміз, бұл барлық тапсырмалардың дұрыс орындалғанын білдіреді. Енді барлық жүйелерден шыға аламыз. Ең алдымен, бұл AWS Management Console. Жоғарғы оң жақтағы пайдаланушы атын басып, ашылмалы тізімдегі Sign out батырмасын басу керек. Әрі қарай зертханалық жұмыстан дұрыс шығуымыз керек. Ол үшін End lab батырмасын басып, зертханалық жұмысты аяқтағанымызды растап, осы терезені жауып, AWS Academy парақшасын жабуға болатындығы туралы хабарды күту керек. Осымен, зертханалық жұмысты талдадық. Сіз оны өзіңіз де дұрыс орындай алдыңыз деп үміттенемін. Сұрақтарыңыз болса жазыңыздар. Назарларыңызға рахмет. Келесі белсенділіктерде кездескенше.
